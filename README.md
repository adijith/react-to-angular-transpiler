# React to Angular Transpiler

A comprehensive tool for transpiling React components to Angular components.

## Features

- Converts React JSX to Angular templates
- Transforms React hooks to Angular lifecycle hooks
- Maps React event handlers to Angular event bindings
- Generates TypeScript, HTML, and CSS files
- Supports both simple and complex React components

## Installation

```bash
# Install Python dependencies
pip install -r requirements.txt

# Install Node.js dependencies
npm install
```

## Usage

```bash
python -m src.transpiler <input_file> <output_directory>
```
